## üîÑ System Workflow

### Student Check-in Process
1. Report to Hostel Office
2. Room allocation confirmation
3. Room inspection & acceptance
4. Key handover
5. Registration in system
6. Welcome kit distribution

### Room Change Process
1. Submit application through system
2. Warden review
3. Room inspection
4. Admin approval
5. Moving assistance coordination
6. Final inspection & confirmation

### Leave Application Process
1. Student submits application
2. Warden reviews & approves/rejects
3. Admin final review
4. Parent notification (if required)
5. System status update
6. Hostel office intimation

### Maintenance Request Process
1. Student submits complaint
2. Office acknowledges receipt
3. Maintenance staff assignment
4. Work completion & inspection
5. Student sign-off
6. System closure
